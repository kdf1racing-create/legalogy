# `legalogy setup`

Initialize `~/.legalogy/legalogy.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
legalogy setup
legalogy setup --workspace ~/.legalogy/workspace
```

To run the wizard via setup:

```bash
legalogy setup --wizard
```
