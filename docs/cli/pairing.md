# `legalogy pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
legalogy pairing list whatsapp
legalogy pairing approve whatsapp <code> --notify
```
