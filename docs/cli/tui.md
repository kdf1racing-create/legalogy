# `legalogy tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
legalogy tui
legalogy tui --url ws://127.0.0.1:18789 --token <token>
legalogy tui --session main --deliver
```
